Automatically delete bbPress spam replies older than X days. 

== Description ==

Use this plugin to automatically detect and remove bbPress replies marked as spam from the database. 

This plugin was developed to help improve database optimization.


= About the Plugin =

http://youtu.be/G2AEf5VEGIk

== Installation ==

1. Upload `bbpress-auto-delete-spam-replies` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
